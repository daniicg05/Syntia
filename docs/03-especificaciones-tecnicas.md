# Especificaciones Técnicas del Proyecto: Syntia

## 1. Arquitectura
- Modelo cliente-servidor con separación frontend / backend.
- **Backend:** Spring Boot, gestión de usuarios, integración con BDNS, motor de priorización.
- **Frontend:** Thymeleaf, responsive, dashboard y panel administrativo.
- **Base de datos:** almacenamiento de usuarios, perfiles, recomendaciones y metadatos de convocatorias.

## 2. Tecnologías Seleccionadas

| Capa | Tecnología |
|------|------------|
| Backend | Java 17+, Spring Boot 3.1+, Spring Security (JWT) |
| Frontend | HTML5, CSS3, JavaScript, Thymeleaf |
| Base de datos | PostgreSQL 17.2 |
| Infraestructura | Servidor en la nube, comunicación HTTPS |

## 3. Estándares
- **Seguridad:** autenticación JWT, cifrado de datos, HTTPS.
- **Desarrollo:** convenciones de Java, validación de entradas, control de acceso por roles.
- **Interfaz:** diseño intuitivo, profesional y responsivo.

## 4. Interfaz de Usuario
- Registro, autenticación y captura de perfil.
- Dashboard interactivo y roadmap estratégico.
- Panel administrativo para supervisión.

## 5. Seguridad
- Control de acceso por roles.
- Cifrado de datos y comunicación segura.
- Prevención de vulnerabilidades comunes.
